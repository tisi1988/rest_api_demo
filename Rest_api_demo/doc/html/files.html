 requestevents.hRequests events declaration. These events are used by the endpoints to notify the application the requests received and their parameters. These messages inherit from the Qt native QEvent class and they're delivered to the application using the Qt's native message system on an asynchronous way. By this way we can ensure when a request is parsed, the endpoints thread goes back to listening for new requests since the message is then handled by another thread (Qt's Event Loop thread)
